文本关键词筛选神器 V4.2 - 使用说明

📌 用途说明：
本程序用于对文本中的关键词进行高亮显示、频率统计、词云生成以及导出分析。
适合政企单位、内容分析、舆情研判、教学科研等场景。

🔐 安全声明：
本程序仅在本地运行，不联网、不上传任何文档，保证数据隐私与安全。
可放心在办公环境中使用。

📦 使用方式：
1. 双击 TextFilterApp.exe 启动程序。
2. 加载关键词文件（.txt），再加载待分析文档（.txt 或 .docx）。
3. 支持导出 Word/CSV、生成词云图，可视化呈现结果。
4. 若有杀毒误报，请设置为信任或添加白名单。
